# Source 3: PRE San Marcos practices

## Sources

A collection of unstructured and very inconsistent PDFs directly from the pratices of the peruvian
preparatory academy "PRE San Marcos", for the semesters between 2010-I and 2022-II

All the PDF's have been extracted using advanced pdf to text tools, and the raw text
outputs are in the `raw_sources` directory

Because we find too much inconsistencies and irregular structures, we opted for using
LLM analysis directly on the files, dividing the contents in chunks.
